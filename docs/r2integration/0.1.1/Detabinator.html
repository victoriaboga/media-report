Constructor

new Detabinator(element)

Source:
  • side-nav/detabinator.js, line 19

Helps to maintain a healthy tabulation within the sidebar when it's open, rather than tabulate the whole page Usage:

const detabinator = new Detabinator(element);
detabinator.inert = true;  // Sets all focusable children of element to tabindex=-1
detabinator.inert = false; // Restores all focusable children of element
